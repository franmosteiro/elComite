R.A.P. (Rely on Android Platform)

¿Desarrollas Android pero no acabas de "rematar"?

¿Tienes controlado el flujo de desarrollo de apps pero te falta salto cualitativo en una app y LA app?

El objetivo de la charla es conseguir la máxima eficiencia posible en el desarrollo de aplicaciones Android:

  • Consumo eficiente de recursos (RAM, network y batería)
  • Código estructurado y limpio (sin dependencias a terceros)
  • Compatibilidad con todas las versiones de Android post Froyo

Veremos cómo desarrollar una aplicación Android utilizando los componentes del propio sistema operativo:

  • AccountManager para gestionar los credenciales de usuario
  • SyncManager para centralizar la lógica de sincronización con el servidor
  • ContentProvider para almecenar datos persistentes
  • Loaders para cargar los datos en la UI

Finalmente,completaremos con una demo, para bucear un poco en el código fuente.

La charla está orientada a gente que ya haya desarrollado con Android puesto que se trataran temas avanzados; aunque tod@s sois bienvenidos, claro :P
